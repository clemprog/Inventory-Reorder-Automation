# Inventory-Reorder-Automation

Problem: Retailers or warehouses forget to reorder low-stock items. Solution: Daily checks stock levels in Google Sheets, sends reorder emails to suppliers, and notifies managers via Slack. Nodes: Cron trigger, Google Sheets read, Function node to filter low stock, SMTP email, Slack notification. Import File: 3_Inventory_Reorder_Automation.json
